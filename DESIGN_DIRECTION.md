# 亀井工務店サイト デザイン方針

## 参照サイト

- [株式会社 古谷野工務店](https://koyanokoumuten.co.jp/)
- [株式会社 福永工務店](https://fukunaga-komuten.com/)
- [株式会社 坂根工務店](https://sakane.co.jp/)
- [奥野工務店](https://okuno-kk.com/)

## 目指す方向

「AI が作った抽象的な企業サイト」ではなく、実際に地域で家を見てきた工務店の気配があるサイトにする。

参考サイトに共通しているのは、派手な装飾ではなく、写真・余白・言葉の具体性・施工事例の見せ方で信頼感をつくっている点。亀井工務店では、ベージュを基調にしながら、木・紙・土・墨のような落ち着いた質感を使い、平塚の暮らしに寄り添う「静かで、温かく、でも古くさくない」見た目を目指す。

キーワードは「住まいを整える」「長く住む」「暮らしを聞く」「現場を見る」。

## 参考サイトから拾う要素

### 古谷野工務店

- 施工事例を中心にした編集的なトップページ。
- 「実績」「住まい手インタビュー」のように、家そのものより暮らしの背景を見せている。
- ナビゲーションは少なく、写真と案件名で世界観を伝える。

亀井工務店に活かすなら、トップに抽象的なサービス説明を並べるより、施工事例や相談内容を先に見せる。事例名も「トイレリフォーム」だけでなく、「夜も使いやすい明るいトイレへ」のように暮らしの変化が分かる名前にする。

### 福永工務店

- 「大切にしていること」として、対話や暮らしの背景をきちんと文章で伝えている。
- 自然素材、木の家、日々のお便りなど、工務店の日常感がある。
- 施工事例の写真と言葉が柔らかく、親しみやすい。

亀井工務店に活かすなら、「提案します」「サポートします」のような汎用語を減らし、「床の段差」「雨の日だけ気になる」「親が使いやすいように」など、相談の入口になる具体語を増やす。

### 坂根工務店

- 創業や歴史、地域性を強く打ち出している。
- 「四季を感じ、自然とともに暮らす」「十人十家」など、短い言葉で思想を示している。
- 施工事例、イベント、ブログ、現場リポートがあり、会社が動いている感じがある。

亀井工務店に活かすなら、会社の歴史や代表の考え、現場で大切にしていることを前に出す。ニュース欄も架空の更新ではなく、現場日記・お知らせ・相談会など、運用できる粒度に絞る。

### 奥野工務店

- 住宅性能、パッシブ換気、モデルハウス、動画など、専門性を見やすく整理している。
- 英語ラベルを使いながらも、本文は生活者に向けた説明になっている。
- Contact / Request / Tel の導線が分かりやすい。

亀井工務店に活かすなら、リフォーム・塗装・断熱・外まわりを「工事メニュー」だけでなく、「どんな困りごとを解決するか」で整理する。問い合わせ導線は、フォーム・電話・写真を送る、の3つが自然。

## ビジュアル方針

### 色

ベージュをベースにする。ただし、全体を淡いベージュ一色にしない。

- 背景: 生成り、薄い砂色、紙色
- 文字: 墨色、濃い茶、黒に近いグレー
- アクセント: 深緑、土色、錆赤を少量
- 避ける色: 明るすぎる青、SaaS 風の紫グラデーション、蛍光色

赤を使う場合は DESIGN.md のような主役の赤ではなく、見出しの小さな印、重要なリンク、下線程度に留める。日本の工務店らしい落ち着きを優先する。

### 写真

写真が最重要。抽象アイコンやプレースホルダーを置くより、粗くても実際の現場写真のほうがよい。

必要な写真:

- 外観、玄関、室内、床、建具、外壁、屋根
- 職人の手元、道具、木材、現場の養生
- Before / After
- 会社やスタッフの自然な写真

写真のトーン:

- 明るすぎる住宅広告風ではなく、自然光で落ち着いた色
- 人がいない完成写真だけでなく、暮らしや手仕事の気配がある写真
- 角丸や過度な影を付けず、紙面に写真を置くように見せる

写真が未準備の場合は、抽象 SVG ではなく「写真準備中」と分かる簡潔な枠にする。無理にイラスト化しない。

### タイポグラフィ

見出しは大きく、短く、左揃え。説明文は詰め込みすぎず、具体的に。

- H1: 1行から2行で強く言い切る
- H2: セクションごとに短い日本語
- 英語ラベルは使いすぎない
- 「Works」「About」だけではなく、日本語の意味を併記する
- 本文は読みやすさ優先。過度な詰め字や装飾フォントは避ける

見出し例:

- 平塚で、手を入れながら長く住める家に。
- まず、家の状態を見ます。
- 図面の前に、暮らしを聞く。
- 雨の日だけ気になる。その違和感から見に行きます。
- 派手さより、毎日が落ち着くこと。

## レイアウト方針

### トップページ構成

1. ファーストビュー
   - 大きな写真または現場感のあるビジュアル
   - 短いコピー
   - 「相談する」「施工事例を見る」「電話する」

2. 亀井工務店の姿勢
   - 「現場を見る」「暮らしを聞く」「必要な工事を整理する」
   - 3つ程度に絞る

3. 施工事例
   - 写真を大きく見せる
   - 工事名、場所の雰囲気、困りごと、施工内容を短く記載
   - カードを均等に並べるだけでなく、1件目を大きく扱う

4. リフォーム・塗装・新築の入口
   - 工事カテゴリではなく、困りごとベースで整理
   - 例: 段差、寒さ、雨漏り、外壁の色あせ、収納不足

5. 会社の人となり
   - 代表の言葉、現場写真、地域性
   - 架空のスタッフ名や実績数は入れない

6. お知らせ
   - 本当に運用できる範囲に絞る
   - 現場日記、相談会、休業日、お知らせ

7. 問い合わせ
   - 電話番号を分かりやすく
   - フォームは「写真添付できる」前提にしたい
   - 相談内容がまとまっていなくてもよいと伝える

### ページ全体のリズム

- 余白はしっかり取るが、テンプレートのような均一カード連続にしない
- セクションごとに写真、文章、リストの見せ方を変える
- カードの角丸、強い影、グラデーションは控える
- 罫線、余白、写真の比率で整理する
- CTA は各ページの最後だけでなく、自然なタイミングで置く

## コンテンツ方針

### 使う言葉

具体的な暮らしの言葉を使う。

- 床の段差
- 雨の日だけ気になる
- 冬の寒さ
- 外壁の色あせ
- 建具が閉まりにくい
- 親が使いやすいように
- 住みながらの工事
- 先に直すところ、後でもよいところ

### 避ける言葉

AIっぽく見えやすい汎用語は減らす。

- 理想の家づくりを全力でサポート
- 最適なプランをご提案
- 高品質なサービス
- お客様満足度
- 幅広いサービス
- 安心安全で快適な住まい

使う場合は、必ず具体例を添える。

## 施工事例の見せ方

施工事例は、見た目の完成写真だけでなく、相談の背景を含める。

各事例に必要な項目:

- タイトル
- 工事種別
- 相談内容
- 工事で行ったこと
- 工夫した点
- Before / After 写真
- 期間の目安
- 関連タグ

タイトル例:

- つまずきやすかった和室の段差をなくす
- 掃除しやすく、明るいトイレへ
- 雨の日に気になっていた外壁まわりを点検・塗装
- 使っていない和室を、収納のある洋室に

## 現在のサイトから直すべき点

- 抽象的な手描き風パネルや CSS 装飾は減らし、写真前提のレイアウトにする
- 赤と黒の強い DESIGN.md 寄せは、日本の工務店サイトとしては少し強すぎるため、アクセントに留める
- 架空感のあるスタッフ名、実績数、お客様の声は使わない
- ニュースは架空の日付を並べず、運用できる内容にする
- 各ページの Hero は英語ラベルよりも、日本語の一言で意味を伝える
- 「サービス」ではなく「困りごと」「施工事例」「相談の流れ」を中心にする

## 最終的な印象

派手なブランドサイトではなく、写真と余白がきれいな工務店サイト。

古い町の工務店らしい安心感は残しつつ、ページの見せ方は現代的。ベージュ基調で温かく、文章は具体的。施工事例を見ると「ここに相談したら、ちゃんと家を見てくれそう」と感じられる状態を目標にする。
