# AIサイト制作指示書：奥野工務店風のスタイリッシュな工務店サイト

## 目的
既存の工務店サイトを、以下の参考サイトのような「上質・余白・写真中心・信頼感・自然素材感」のあるデザインに刷新してください。

参考サイト： https://okuno-kk.com/

ただし、参考サイトを丸コピーしないでください。レイアウト、余白感、写真の見せ方、情報設計、トーン、アニメーションの方向性を参考にしつつ、独自のデザインとして実装してください。

---

## 参考サイトから取り入れたい雰囲気

- 高級感があるが、派手すぎない
- 住宅・工務店らしい安心感がある
- 写真を大きく使い、暮らしのイメージが伝わる
- 白・生成り・淡いベージュ・木目系の落ち着いた配色
- 文字量を詰め込みすぎず、余白を広く取る
- 英語見出しと日本語見出しを組み合わせる
- 「施工事例」「家づくりの考え方」「会社情報」「お問い合わせ」へ自然に誘導する
- スクロールしていくと各セクションがゆっくり現れるような上品な動き

---

## 技術スタック

以下の構成で実装してください。

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- lucide-react
- shadcn/ui は必要に応じて使用
- レスポンシブ対応必須
- SEOを意識したHTML構造

出力先は `/kamei` 以下にしてください。

## カラーパレット

Tailwindで以下のような色味を使ってください。

- Base: `#F7F3EA` / `#FAF8F3` / `#FFFFFF`
- Text: `#1F2933` / `#2D2A26`
- Sub Text: `#6B6259`
- Accent: `#8A6A43` / `#9B7A4E`
- Dark Section: `#1E1E1B`
- Border: `#E5DED2`

背景は真っ白だけでなく、薄い生成り・ベージュを使い分けてください。

## フォント方針

### 日本語

- `Noto Serif JP` または `Noto Sans JP`
- 見出しは少し品のある明朝寄りでもよい
- 本文は可読性を優先

### 英語見出し

- `Cormorant Garamond`
- `Playfair Display`
- `Inter`

英語の小見出しを薄く入れ、日本語見出しを主役にしてください。

例：

```tsx
<p className="text-sm tracking-[0.3em] uppercase text-stone-500">About us</p>
<h2 className="mt-3 text-3xl md:text-5xl font-serif tracking-wide">私たちの家づくり</h2>
```

## 1. Header

### 要件

- 左上にロゴ
- 右側にナビゲーション
- PCでは横並び
- スマホではハンバーガーメニュー
- スクロールすると背景が半透明の白になり、少しぼかしを入れる
- CTAボタンとして「お問い合わせ」「資料請求」を右側に配置

### ナビゲーション例

- Home
- About
- Works
- Service
- Blog
- Company
- Contact

### アニメーション

- 初回表示時に上からふわっと表示
- スクロール時に影を軽く付与

---

## 2. Hero

### 目的

ファーストビューで「この会社に頼むと、丁寧で上質な家を建ててくれそう」と感じさせる。

### 要件

- 画面いっぱいに近い高さ
- 住宅写真を大きく使用
- 画像は1枚固定
- 中央または左下にキャッチコピー
- 縦書き風または余白を活かしたレイアウトも可

### キャッチコピー例

```txt
人にこたえる、
永く心地よい住まい。
```

または

```txt
暮らしに寄り添い、
時を重ねる家づくり。
```

### サブコピー例

```txt
自然素材のぬくもりと、確かな性能。
地域に根ざした工務店として、家族のこれからを支える住まいをつくります。
```

### アニメーション

- 背景画像をゆっくりズームイン
- テキストを0.2秒ずつ遅延してフェードイン
- Scroll Down 表示を下部に配置し、上下にゆっくり動かす

## 3. News

### 要件

- Hero直下にニュース一覧を配置
- 3件表示
- 日付、カテゴリ、タイトルを表示
- 右側または下部に `view more`

### デザイン

- 罫線を細く使う
- 余白を広く
- ホバー時にタイトルが少し右に動く

## 4. Concept / About

### 目的

会社の家づくりの考え方を伝える。

### 要件

- 左にテキスト、右に写真
- もしくは写真を背景にしてテキストを重ねる
- 英語見出し `About us`
- 日本語見出しは2行程度

### 見出し例

```txt
家族の時間を、
静かに支える家づくり。
```

### 本文例

```txt
私たちは、見た目の美しさだけでなく、毎日の暮らしやすさ、住み続ける安心感、年月を重ねても飽きのこない素材感を大切にしています。地域の気候や土地の特性を読み取り、一棟一棟に向き合いながら、長く愛される住まいを提案します。
```

### アニメーション

- セクションが画面に入ったら、画像は右からフェードイン
- テキストは下からフェードイン
- 写真に軽いパララックス効果

## アニメーション指定

Framer Motionを使って、以下の動きを実装してください。

### 基本方針

- 派手に動かさない
- ゆっくり、上品に
- スクロールに合わせて自然に現れる
- 住宅サイトなので信頼感を損なわない

### 実装してほしい動き

1. ページ初回表示時のHeroテキストフェードイン
2. Hero画像のゆっくりズーム
3. Headerのスクロール時背景変化
4. 各セクションのfade-up
5. Worksカードのホバー拡大
6. CTAボタンのホバー時の矢印移動
7. Scroll Downの軽い上下アニメーション
8. 画像セクションの軽いパララックス

### Framer Motion例

```tsx
const fadeUp = {
  hidden: { opacity: 0, y: 40 },
  visible: {
    opacity: 1,
    y: 0,
    transition: { duration: 0.8, ease: [0.22, 1, 0.36, 1] }
  }
}
```

```tsx
<motion.div
  initial="hidden"
  whileInView="visible"
  viewport={{ once: true, amount: 0.3 }}
  variants={fadeUp}
>
  ...
</motion.div>
```

## AIっぽさを消すための具体指示

以下を必ず守ってください。

- すべてのセクションを同じカード形状にしない
- 余白の取り方に強弱をつける
- 見出し文をテンプレート的にしない
- 画像の縦横比をセクションごとに変える
- 左右交互レイアウトを単純に繰り返さない
- ボタン文言をすべて同じにしない
- 実在する工務店サイトらしい情報量にする
- セクション間に十分な余白を入れる
- 完璧に整いすぎたグリッドではなく、写真の大きさに少し変化をつける
- 装飾よりも写真・余白・タイポグラフィで上質さを出す

## 画像について

実画像がない場合は、以下のようなプレースホルダーを使ってください。

- 外観写真
- 木の質感があるリビング
- 職人が作業している写真
- 家族が過ごすダイニング
- モデルハウス外観
- 施工事例の室内写真

画像パスは一旦 `/images/...` で仮置きしてください。
あとから差し替えやすいよう、画像データは配列で管理してください。

## データ管理

施工事例、ニュース、ブログ、強みは `/lib/data.ts` に配列として定義してください。

例：

```tsx
export const works = [
  {
    title: '木のぬくもりに包まれる平屋',
    description: '家族が自然と集まる、開放的なリビングのある住まい。',
    image: '/images/work-01.jpg',
    tags: ['平屋', '自然素材', '注文住宅'],
  },
]
```

## レスポンシブ要件

### PC

- 最大幅は `1200px〜1280px`
- 写真とテキストの横並びを活かす
- Heroは大きく、余白を広く

### タブレット

- 2カラムを適宜1カラムへ
- 画像は横長を維持

### スマホ

- 1カラム
- ナビゲーションはハンバーガー
- CTAは押しやすいサイズ
- 文字サイズは小さくしすぎない
- Heroテキストは2〜3行で読みやすく

---

## SEO / アクセシビリティ

- `h1` はHeroに1つだけ
- セクションごとに適切な `h2` を使用
- 画像には必ず `alt` を入れる
- ボタン・リンクは意味のある文言にする
- コントラストを確保する
- meta title / description を設定

### meta例

```txt
title: 亀井工務店 | 暮らしに寄り添う家づくり

description: 地域に根ざした工務店として、自然素材と確かな施工で、長く心地よく暮らせる住まいをご提案します。
```

---

## 実装時の注意

- 参考サイトのHTMLやCSSをコピーしない
- 画像・文章・ロゴを無断使用しない
- あくまでデザインの方向性だけを参考にする
- 既存サイトの内容がある場合は、その情報を優先して反映する
- 不明な情報は仮テキストでよいが、あとから差し替えやすくする
- 最後に、実装したファイル一覧と起動方法を提示する
